When entering variations into the program, enter the moves without the move number:

	WRONG:
	
	1. e4 e5 2. Nh3 f5 ...

	RIGHT:

	e4 e5 Nh3 f5 ...

Case DOES matter when entering variation names and moves.

If changing/adding variations directly through the text file, add it on a new line in this form:

	[variation name] : [sequence of moves]

	i.e.

 	Open Spanish : e4 e5 Nf3 Nc6 Bb5 a6 Ba4 Nf6 O-O Nxe4

Please do not add any lines not of this form, it may (temporarily) break the program.
